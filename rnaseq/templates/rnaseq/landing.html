Welcome to the RNA-seq Analysis Portal by Rockefeller University CCTS Research Bioinformatics.

This is an on-going effort to help investigators perform their downstream RNA-seq analyses.

At the moment, the system assumes the mapping (tophat, STAR) and feature summarization (HTSeq, featureCounts) has been applied to the RNA-seq data.

Linear model gene-level differential expression analysis is currently supported (voom: reference to https://genomebiology.biomedcentral.com/articles/10.1186/gb-2014-15-2-r29). In development is an analysis that supports the negative binomial model (DESeq2, edgeR).

To gain access to the system please contact liangy@rockefeller.edu.
